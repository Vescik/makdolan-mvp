# AI Brain Index

This index points to the most important AI Brain and repository context files.

Regenerate the local repository index with:

```bash
npm run brain:index
```

Search AI Brain and project context with:

```bash
npm run brain:search -- "query"
```

Create pre-implementation impact guidance with:

```bash
npm run brain:impact -- "change description"
```

Run local AI Brain governance health checks with:

```bash
npm run brain:health
```
