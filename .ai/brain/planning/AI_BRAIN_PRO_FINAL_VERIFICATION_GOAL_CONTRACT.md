# AI Brain Pro Final Verification Goal Contract

## Goal Title

`AI Brain Pro: verify end-to-end SDLC workflow`

## Objective

Verify that AI Brain Pro supports the Makdolan agent lifecycle from DISCOVER through ITERATE using only local deterministic tooling, documentation, validation evidence, and disabled automation templates.

## Done When

- `npm run brain:index` refreshes the repository index successfully.
- `npm run brain:search` returns useful results for at least three realistic queries.
- `npm run brain:context` creates a usable sample feature context pack.
- `npm run brain:impact` creates a usable heuristic impact analysis.
- A final verification report exists at `.ai/brain/planning/AI_BRAIN_PRO_FINAL_VERIFICATION_REPORT.md`.
- Validation evidence includes `bash scripts/diff-gate.sh`.
- Maker/checker dry run evidence is recorded.
- AI Brain memory is updated with the final integration result.

## Allowed Scope

- Allowed files:
  - `.ai/brain/planning/AI_BRAIN_PRO_FINAL_VERIFICATION_GOAL_CONTRACT.md`
  - `.ai/brain/planning/AI_BRAIN_PRO_FINAL_VERIFICATION_REPORT.md`
  - `.ai/brain/context-packs/*.md` generated by AI Brain scripts
  - `.ai/brain/index/*` generated by `npm run brain:index`
  - `.ai/brain/memory/implementation-history.md`
- Allowed commands:
  - `npm run brain:index`
  - `npm run brain:search -- "query"`
  - `npm run brain:context -- "task" --phase=DISCOVER --summary="summary"`
  - `npm run brain:impact -- "change description"`
  - `npm run brain:memory:update`
  - `bash scripts/diff-gate.sh`
  - `codex review --uncommitted`
  - `git diff --check`
- Allowed behavior changes:
  - Documentation and generated AI Brain planning/index/context artifacts only.

## Forbidden Scope

- App source behavior changes under `app/` or `src/`.
- Dependency installation, removal, or upgrade.
- CI, deployment, signing, release, database, authentication, payment, or production automation changes.
- Reading, copying, or persisting secrets, credentials, `.env.local` values, tokens, authorization headers, or private user data.
- Enabling recurring automations or dangerous hooks.

## Stop Conditions

Stop before:

- Changing app behavior.
- Enabling automations.
- Adding dependencies.
- Reading or exposing secrets.
- Treating heuristic search or impact output as certainty.
- Marking verification complete without validation evidence.
