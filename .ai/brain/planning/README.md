# Planning

This folder stores AI Brain Pro discovery reports, phase plans, and implementation plans.

Use it when:

- A phase needs clear scope and acceptance criteria.
- A change affects multiple files, platforms, or validation gates.
- There is a meaningful architecture, workflow, data, or release-risk decision to make before editing.

Current artifacts:

- `AI_BRAIN_PRO_DISCOVERY_REPORT.md`: Phase 0 discovery report for the AI Brain Pro setup.

Planning artifacts should include scope, affected files, implementation steps, validation commands, risks, rollback notes, and a definition of done.
