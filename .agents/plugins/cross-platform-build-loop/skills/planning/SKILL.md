---
name: planning
description: Convert discovery output into a small, testable implementation plan for Codex.
---

Create a plan with:

- Goal and acceptance criteria.
- Affected files/modules.
- Implementation steps.
- Test and build commands.
- Platform matrix: iOS, Android, Web.
- Security/privacy notes.
- Rollback strategy.
- Definition of done.

Prefer vertical slices over broad rewrites.
