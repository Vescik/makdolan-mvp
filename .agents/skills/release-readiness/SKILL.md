---
name: release-readiness
description: Check whether a change or release is ready for iOS, Android, and Web deployment.
---

Review:

- CI status and required checks.
- Versioning/build numbers.
- Environment variables and secrets.
- Store signing placeholders.
- Web hosting build output.
- Changelog/release notes.
- Rollback and hotfix path.

Do not deploy or publish without explicit approval.
